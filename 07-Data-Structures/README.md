## 📖 学习内容

### 1. 线性数据结构

#### 数组（Array）
**特点：**
- 连续存储
- 随机访问
- 查找快（O(1)），插入和删除慢（O(n)）

**应用：**
- 缓存
- 数据排序
- 二分查找

#### 链表（LinkedList）
**特点：**
- 离散存储
- 顺序访问
- 查找慢（O(n)），插入和删除快（O(1)）

**类型：**
- 单向链表
- 双向链表
- 循环链表

**应用：**
- LRU 缓存
- 撤销/重做功能
- 操作系统进程管理

#### 栈（Stack）
**特点：**
- LIFO（后进先出）
- 只能在一端操作

**应用：**
- 括号匹配
- 函数调用栈
- 表达式求值
- 深度优先搜索

#### 队列（Queue）
**特点：**
- FIFO（先进先出）
- 只能在两端操作

**变种：**
- 循环队列
- 优先级队列
- 双端队列

**应用：**
- 任务调度
- 消息队列
- 广度优先搜索
- 操作系统缓冲区

### 2. 树型数据结构

#### 二叉树（Binary Tree）
**基本概念：**
- 节点、边、叶子节点、内部节点
- 深度、高度、度

**遍历方式：**
- 前序遍历（Pre-order）
- 中序遍历（In-order）
- 后序遍历（Post-order）
- 层序遍历（Level-order）

**特殊二叉树：**
- 完全二叉树
- 满二叉树
- 平衡二叉树

#### 二叉搜索树（BST）
**特点：**
- 左子树 < 父节点 < 右子树
- 中序遍历得到有序序列
- 查找、插入、删除时间复杂度 O(log n)

**缺点：**
- 可能退化为链表（O(n)）

#### 平衡二叉树（AVL 树）
**特点：**
- 自动保持平衡
- 任何节点的左右子树高度差不超过 1
- 查找、插入、删除时间复杂度 O(log n)

**应用：**
- 数据库索引
- 文件系统

#### 红黑树（Red-Black Tree）
**特点：**
- 自平衡二叉搜索树
- 放松的平衡条件
- 更高的插入和删除性能

**应用：**
- Java 的 TreeMap、TreeSet
- Linux 内核

### 3. 哈希表（Hash Table）

**原理：**
- 哈希函数将键映射到数组索引
- 通过数组随机访问实现高效查询

**冲突解决方法：**
- 链地址法（拉链法）
- 开放地址法
- 双重哈希

**性能：**
- 平均时间复杂度 O(1)
- 最坏情况 O(n)

**应用：**
- Java HashMap、HashSet
- 缓存实现
- 数据库索引

### 4. 图（Graph）

**基本概念：**
- 顶点（Vertex）、边（Edge）、度（Degree）
- 有向图、无向图、加权图
- 连通图、强连通图

**表示方法：**
- 邻接矩阵
- 邻接表

**遍历：**
- 深度优先搜索（DFS）
- 广度优先搜索（BFS）

**经典算法：**
- Dijkstra 最短路径
- Kruskal 最小生成树
- 拓扑排序

**应用：**
- 社交网络
- 地图导航
- 推荐系统

## 💡 时间复杂度分析

**常见数据结构操作的时间复杂度：**

| 数据结构 | 查找 | 插入 | 删除 | 空间 |
|--------|------|------|------|------|
| 数组 | O(1) | O(n) | O(n) | O(n) |
| 链表 | O(n) | O(1) | O(1) | O(n) |
| 栈 | O(n) | O(1) | O(1) | O(n) |
| 队列 | O(n) | O(1) | O(1) | O(n) |
| 二叉搜索树 | O(log n) | O(log n) | O(log n) | O(n) |
| 哈希表 | O(1) | O(1) | O(1) | O(n) |
